# Claude 101: 入门教程

**课程概述**

Claude 101 是 Anthropic 的入门课程，专为 Claude 新用户设计。本课程将教你如何使用 Claude 完成日常工作任务。

## 第一章：什么是 Claude?

Claude 是 Anthropic 开发的人工智能助手，擅长：
- 对话和理解上下文
- 写作和编辑
- 分析和总结
- 编程和技术任务
- 复杂推理

### 核心特性
1. **安全性**：Claude 经过 Constitutional AI 训练，注重安全性和有用性
2. **长上下文**：支持高达200K tokens的上下文窗口
3. **多模态**：可以理解图片和文本
4. **实时协作**：支持Projects和Artifacts功能

## 第二章：你的第一次对话

**开始使用 Claude**：

1. 访问 https://claude.ai
2. 注册账号（支持Google登录）
3. 开始新对话

### 示例Prompt：

```
你好，Claude！我是市场营销专员，希望学习如何使用AI提升工作效率。
你能帮我制定一个学习计划吗？
```

**Claude的回复特点**：
- 友好、专业
- 提供结构化建议
- 可以根据你的角色调整回答

## 第三章：获取更好的结果

**Prompt工程基础**：

### 原则1：具体明确
❌ 不好："帮我写点东西"
✅ 好："帮我写一封100字的邮件，邀请客户参加产品发布会"

### 原则2：提供上下文
```
我是[角色]，正在做[任务]，目标是[目标]。
请[具体行动]，要求：[约束条件]
```

### 原则3：使用示例
提供一个例子，Claude会理解你的期望格式和风格。

## 第四章：Claude 桌面应用

**Claude Desktop App** 功能：
- 聊天模式：正常对话
- 协作模式（Cowork）：与Claude一起工作
- 代码模式（Code）：编程助手

### Artifacts功能
当Claude生成：
- 长文档
- 代码
- 网页内容

会弹出独立窗口，方便编辑和迭代。

## 第五章：Projects（项目）

**Projects** 让你可以：
1. 上传文档作为知识库
2. 设置自定义指令
3. 在同一上下文中多轮对话

### 示例用例
- 将所有产品文档上传到Project
- 让Claude基于这些文档回答问题
- 保持对话上下文

## 第六章：Skills（技能）

**Skills** 是Claude的插件系统：
- 搜索网络
- 生成图片
- 分析数据
- 自定义工作流

### 启用Skills
在对话中，Claude会建议相关Skills，或你手动启用。
