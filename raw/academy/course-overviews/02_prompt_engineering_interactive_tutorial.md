## 课程描述

A comprehensive step-by-step guide to key prompting techniques. This course is designed for interactive learning with hands-on practice.

**Also available**: [AWS Workshop version](https://catalog.us-east-1.prod.workshops.aws/workshops/0644c9e9-5b82-45f2-8835-3b5aa30b1848/en-US)

**Note**: This is the same content as `anthropics/prompt-eng-interactive-tutorial` (separate repo)

## 学习目标

After completing this course, you will be able to:
- Master key prompting techniques for Claude
- Understand clarity, examples, and XML structuring
- Apply role prompting and chain-of-thought techniques
- Use the Claude Console prompting tools effectively

## 课程结构（9 个章节 + 附录）

### Beginner Level (Chapters 1-3)
- **Chapter 1**: Introduction to prompt engineering
- **Chapter 2**: Basic prompting techniques
- **Chapter 3**: Clarity and precision in prompts

### Intermediate Level (Chapters 4-7)
- **Chapter 4**: Using examples effectively
- **Chapter 5**: XML structuring for complex prompts
- **Chapter 6**: Role prompting techniques
- **Chapter 7**: Chain-of-thought prompting

### Advanced Level (Chapters 8-9)
- **Chapter 8**: Advanced prompting strategies
- **Chapter 9**: Real-world applications

### Appendix
- Additional resources and references
- Troubleshooting guide
