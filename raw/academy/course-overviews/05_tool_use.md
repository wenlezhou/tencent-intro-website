# Tool Use#

**课程来源**: Anthropic 官方 GitHub 仓库
**仓库路径**: `tool_use/README.md`
**课程类型**: 工具使用交互式教程（6 个 Notebook）

---

## 课程描述

Learn everything you need to know to implement tool use successfully in your workflows with Claude. Across 6 lessons.

## 学习目标

After completing this course, you will be able to:
- Understand Claude's tool use capabilities
- Implement your first simple tool
- Force JSON output with tool use
- Build complete tool use workflows
- Handle tool choice and multi-tool scenarios
- Build chatbots with multiple tools

## 课程章节（6 个 Notebook）

### Lesson 1: Tool Use Overview (`01_tool_use_overview.ipynb`)
- Understanding tool use concepts
- How Claude uses tools
- Basic tool use workflow

### Lesson 2: Your First Simple Tool (`02_your_first_simple_tool.ipynb`)
- Creating your first tool
- Defining tool schemas
- Handling tool execution

### Lesson 3: Structured Outputs (`03_structured_outputs.ipynb`)
- Forcing JSON with tool use
- Combining tool use with structured outputs
- Best practices for reliable outputs

### Lesson 4: Complete Workflow (`04_complete_workflow.ipynb`)
- Building a complete tool use workflow
- Handling multi-turn tool conversations
- Error handling and retries

### Lesson 5: Tool Choice (`05_tool_choice.ipynb`)
- Allowing Claude to choose tools
- Implementing dynamic tool selection
- Optimizing tool choice strategies

### Lesson 6: Chatbot with Multiple Tools (`06_chatbot_with_multiple_tools.ipynb`)
- Building a chatbot with multiple tools
- Managing complex tool interactions
- Production-ready implementations
