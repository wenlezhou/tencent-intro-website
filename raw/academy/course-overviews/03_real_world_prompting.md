**课程来源**: Anthropic 官方 GitHub 仓库

## 课程描述

Learn how to incorporate prompting techniques into complex, real world prompts. Designed for experienced developers who have completed the Prompt Engineering Interactive Tutorial.

**Also available**: [Google Vertex version](https://github.com/anthropics/courses/tree/vertex/real_world_prompting)

## 学习目标

After completing this course, you will be able to:
- Apply prompting techniques to real-world scenarios
- Build complex, multi-step prompts
- Handle edge cases and unexpected inputs
- Optimize prompts for production use

## 课程章节（5 个 Notebook）

### 1. Prompting Recap (`01_prompting_recap.ipynb`)
- Review of key prompting concepts
- Refresher on basic techniques
- Preparing for advanced applications

### 2. Medical Prompt (`02_medical_prompt.ipynb`)
- Walking through a medical domain prompt
- Handling specialized terminology
- Ensuring accuracy and safety

### 3. Prompt Engineering (`03_prompt_engineering.ipynb`)
- The prompt engineering process
- Iterative refinement
- Testing and evaluation

### 4. Call Summarizer (`04_call_summarizer.ipynb`)
- Building a call summarization prompt
- Handling long-form content
- Extracting structured information

### 5. Customer Support AI (`05_customer_support_ai.ipynb`)
- Creating a customer support bot prompt
- Handling multi-turn conversations
- Managing tone and personality

## 技术要求

- **Python 环境**: Requires Python 3.7+
- **Jupyter Notebook**: Required to run the tutorials
- **Anthropic API Key**: You need an API key to run the examples
