# Anthropic API Fundamentals

**课程来源**: Anthropic 官方 GitHub 仓库
**仓库路径**: `anthropic_api_fundamentals/README.md`
**课程类型**: 交互式 Jupyter Notebook 教程

---

## 课程描述

A series of notebook tutorials that cover the essentials of working with Claude models and the Anthropic SDK. This course provides hands-on experience with Claude API fundamentals.

## 学习目标

After completing this course, you will be able to:
- Get started with Anthropic API and SDK
- Understand the messages format for Claude interactions
- Compare capabilities and performance of the Claude model family
- Tune model parameters for optimal results
- Work with streaming responses
- Use Claude's vision capabilities

## 课程章节（6 个 Notebook）

### 1. Getting Started (`01_getting_started.ipynb`)
- Getting an API key
- Making simple API requests
- Understanding basic authentication

### 2. Messages Format (`02_messages_format.ipynb`)
- Understanding the messages API structure
- Multi-turn conversations
- System prompts and user messages

### 3. Models (`03_models.ipynb`)
- Comparing capabilities and performance of the Claude model family
- Choosing the right model for your use case
- Understanding model trade-offs (cost vs. capability)

### 4. Parameters (`04_parameters.ipynb`)
- Understanding model parameters (temperature, max_tokens, etc.)
- Tuning parameters for optimal results
- Best practices for parameter selection

### 5. Streaming (`05_streaming.ipynb`)
- Working with streaming responses
- Handling streaming in different programming languages
- Building responsive applications with streaming

### 6. Vision (`06_vision.ipynb`)
- Vision prompting with Claude
- Sending images to Claude
- Understanding vision capabilities and limitations

## 技术要求

- **Python 环境**: Requires Python 3.7+
- **Anthropic SDK**: Install via `pip install anthropic`
- **Jupyter Notebook**: Required to run the tutorials
- **API Key**: You need an Anthropic API key

- This course favors using Claude 3 Haiku (lowest-cost model) to keep API costs down for students.
